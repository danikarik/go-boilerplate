# Porject Boilerplate

Personal project boilerplate

## Components

- [Golang](https://golang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Vue.js](https://vuejs.org/)

## Usage

Build

```sh
make
```

Run

```sh
make run
```

Live

```sh
make watch
```

```sh
make dev
```
